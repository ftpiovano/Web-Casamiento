# Specification: Enhanced Event Interaction (Maps & Calendar)

## Overview
Enable real-world navigation and calendar integration for guests. The "Where & When" section should provide direct links to Google Maps and downloadable calendar events (.ics) with accurate location and time data for both Argentina and Brazil celebrations.

## Features
1. **Accurate Map Links:** Buttons that open the specific Google Maps URLs for the Hotel in Baixio (BR) and Club de Campo San Diego (AR).
2. **Calendar Events (.ics):** Downloadable files that include the event name, date, time, and the physical address as the location.
3. **Dynamic Configuration:** All location data (name, address, link, date/time) is managed centrally in site.config.ts.
