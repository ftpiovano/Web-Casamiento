# Specification: MVP Foundation and Core Single-Page Layout

## Overview
This track focuses on establishing the project's technical foundation and building the core single-page landing site for the wedding.

## Features
1. **Next.js Scaffolding:** Initialize app with TypeScript, Tailwind, and folder structure.
2. **Site Configuration:** Implement site.config.ts for centralized management.
3. **Responsive Navigation:** Sticky navbar with anchor links and mobile drawer.
4. **Hero & Countdown:** Visual entry point with names, date, and live timer.
5. **Event Details:** Ceremony and Reception sections with map links and .ics generation.
6. **Bilingual Welcome:** Content layout supporting two languages.
7. **Gift Registry Prototype:** Grid display of items with placeholders.
8. **RSVP & Guestbook Logic:** Basic forms with LocalStorage persistence.
