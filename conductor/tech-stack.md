# Tech Stack

## Frontend Framework
- **Next.js 14+ (App Router):** Provides a robust foundation with excellent performance and SEO capabilities.
- **React:** For building interactive UI components like the countdown, RSVP form, and gift registry.
- **TypeScript:** To ensure type safety and improve maintainability.

## Styling
- **Tailwind CSS:** For rapid, utility-first styling consistent with our neutral and airy design requirements.
- **Lucide React:** For a consistent and lightweight icon set.
- **Framer Motion:** To implement the "subtle motion" requirements (gentle fade/slide-ins on scroll).

## Fonts
- **Next/Font (Google Fonts):**
    - **Playfair Display:** Serif font for headings.
    - **Inter:** Sans-serif font for body text.

## State Management & Storage
- **React Hooks (useState, useEffect):** For local component state.
- **LocalStorage API:** For persisting RSVP entries and guestbook messages in the browser during the prototype phase.

## Development Tools
- **ESLint & Prettier:** For code quality and consistent formatting.
- **TypeScript:** For static type checking.
