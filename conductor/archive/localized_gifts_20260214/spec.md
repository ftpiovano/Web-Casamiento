# Specification: Advanced Localization and Dynamic Gift Registry

## Overview
Expand the site's localization to support three regions (BR, AR, EN) and implement a dynamic gift registry where prices and descriptions are customized for each region.

## Features
1. **Triple-Region Support:** Support for Portuguese (BR), Spanish (AR), and English (International).
2. **Dynamic Localized Gifts:** Centralized gift list in site.config.ts with region-specific pricing and text.
3. **Region-Aware UI:** All components pull strings and currency symbols from the active region's configuration.
4. **Enhanced Navigation:** Updated region switcher in the Navbar.
