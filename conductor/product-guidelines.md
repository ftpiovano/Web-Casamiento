# Product Guidelines

## Brand Voice & Tone
- **Formal & Romantic:** The language should be elegant and traditional, using warm but respectful phrasing for invitations and descriptions.
- **Invitational:** Every section should feel like an invitation to join a celebration.
- **Bilingual Consistency:** Ensure that the tone remains consistent across both provided languages (e.g., PT / ES).

## Visual Identity
- **Modern Ethereal Aesthetic:** Prioritize white space ("breathing room") to create an airy, uncluttered feel.
- **Palette:** 
    - Background: Warm off-white (#faf8f5).
    - Text: Charcoal (#222) for high readability with a soft touch.
    - Accents: Sand/beige and one configurable primary color for subtle highlights.
- **Typography:**
    - Headings: Serif (e.g., Playfair Display) to convey tradition and elegance.
    - Body: Sans-serif (e.g., Inter) for modern clarity and ease of reading.
- **UI Elements:**
    - Use very thin, subtle dividers.
    - Soft rounded corners on cards and buttons.
    - Extremely light shadows to create depth without bulk.

## Motion & Interaction
- **Gentle Transitions:** Use subtle fade-ins and slide-ups as the user scrolls.
- **Hover States:** Links and buttons should have soft transitions (e.g., slight color shift or underline) rather than abrupt changes.
- **Smooth Navigation:** Ensure all anchor links utilize smooth scrolling.

## Content Guidelines
- **Placeholders:** Use descriptive placeholders (e.g., [Bride Name], [Groom Name], [Ceremony Address]) to make it obvious where the user needs to add their own content.
- **Multi-region UX:** Present language options clearly (BR, AR, EN), ensuring that switching regions is seamless, updates currency and pricing, and doesn't disrupt the scroll position.
